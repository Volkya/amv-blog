== Description ==

CGS Travel Agency is flexible, fast, lightweight and modern multipurpose theme and suitable for personal blog, portfolio, WooCommerce stores, business websites and niche-based sites (like Cafe, Spa, Charity, Yoga, Wedding, Dentist, Education etc) as well. Works perfectly with Elementor plus other major page builders so you can create any layout . The theme is responsive, Gutenberg compatible, SEO friendly, translation ready and major WordPress plugins compatible.

Demo URL: https://www.cgsthemes.com/product/free-travel-agency/
Read documentation: https://www.cgsthemes.com/documentation/travel-agency-free-theme-documentation/
Get support: https://www.cgsthemes.com/contact-us/


== Installation ==

= Through Dashboard =

1. Log in to the WordPress Dashboard.
2. Go to Appearance -> Themes.
3. Click on Add New Theme.
4. Click on Upload Theme button.
5. Click on Choose File button and select cgs-travel-agency.zip then click on Install Now button and wait for the install to complete.
6. After the theme installed successfully, click on Activate button.

= Manual Installation using FTP =

1. Unzip cgs-travel-agency.zip to your Desktop.
2. Use an FTP client to access your host web server. Go to /wp-content/themes/
3. Upload theme to /wp-content/themes/.
4. Go to Appearance -> Themes and activate theme from Dashboard.
